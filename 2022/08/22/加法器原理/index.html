加法器原理

一. 简介

  1. 加法器，顾名思意就是实现加法运算的一个装置，在计算机中，加法器是最基本的运算装置，因为在计算机中减法，乘法，除法都可以转化为加法。
  2. 加法器分为半加器和全加器，半加器指的是只包含两个加数作为输入，一个和数和一个进位作为输出，而全加器是在半加器的基础上加上了一个进位作为输入。

二. 实现原理

    0 + 1 = 0 1
    1 + 0 = 0 1
    1 + 1 = 1 0
